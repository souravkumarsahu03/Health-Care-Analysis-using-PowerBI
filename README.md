# Health Care Analysis using PowerBI

### Introduction
Health care is a vital industry focused on improving, maintaining, and restoring the health of individuals through medical services, prevention programs, diagnostics, and treatment solutions.
Â Worldwide, health care systems face growing challenges like workforce shortages, rising costs, and the need for increased capacity and innovation.

### Objective
Centralize Data: To build a single, interactive dashboard in Power BI.
Track Key Metrics: To monitor KPIs like Total Revenue, Patient Volume.
Identify Trends: To find insights on admission patterns, high-revenue medical conditions, and operational efficiency, and Average Length of Stay.

### Tools and Approach
Tool : This entire project was built using Microsoft Power BI.
Process:
Data Connection: Loaded the hospital's raw healthcare data.
Data Modeling: Cleaned the data and created DAX measures to calculate KPIs like Total Revenue and Average Length of Stay.
Visualization: Designed this interactive dashboard with simple filters for Year and Gender.
